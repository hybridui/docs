# Locales

### Arabe

By setting the locale to "ar", the date picker will use the formatting, language, and other conventions of the Arabic locale.

<HyDatepicker
  locale="ar"
  fieldFormat="DD/MM/YYYY"
  dateValue="20/11/2024"
  dateplaceholder="20.11.2024"
></HyDatepicker>

### Chinese

Similar to "Arabe", "Chinese" is another locale option for the component. Setting the locale to "zh" applies Chinese conventions to the date picker.

<HyDatepicker
  locale="zh"
  fieldFormat="DD/MM/YYYY"
  dateValue="20/11/2024"
  dateplaceholder="20.11.2024"
></HyDatepicker>

## Display mode

This section demonstrates how to change the display mode of the HyDatepicker component. There are different modes such as "month" and "year", which change how the date picker is displayed.

### Day

This is a display mode for the component.This is the default, it allowi the users to select a specific day, which allow also change month and year.

<HyDatepicker
  fieldFormat="DD/MM/YYYY"
  dateValue="20/11/2024"
  dateplaceholder="20.11.2024"
></HyDatepicker>

### Month

This is a display mode for the component. When the mode is set to "month", the date picker shows the months, allowing users to select a specific month.

<HyDatepicker
  fieldFormat="DD/MM/YYYY"
  dateValue="20/11/2024"
  dateplaceholder="20.11.2024"
  mode="month"
></HyDatepicker>

### Year

Similar to "month", "year" is another display mode for the component. In this mode, the date picker allows users to select a specific year.

<HyDatepicker
  fieldFormat="DD/MM/YYYY"
  dateValue="20/11/2024"
  dateplaceholder="20.11.2024"
  mode="year"
></HyDatepicker>
